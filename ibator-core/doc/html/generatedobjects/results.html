Using the Ibator Generated Objects

Ibator generates these types of objects:

  1. Java Model Classes
  2. SQL Map Files
  3. Java DAO Classes (optional)
  4. A class for use in the xxxByExample methods. See the following pages for information about that class:
    • Example Class Usage Notes
    • Extending the Example Classes

The individual pages describe these objects, and their usage.
