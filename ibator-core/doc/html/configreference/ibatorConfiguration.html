The <ibatorConfiguration> Element

The <ibatorConfiguration> element is the root element of an Ibator configuration file. The file should contain the following DOCTYPE: 

<!DOCTYPE ibatorConfiguration PUBLIC
  "-//Apache Software Foundation//DTD Apache iBATIS Ibator Configuration 1.0//EN"
  "http://ibatis.apache.org/dtd/ibator-config_1_0.dtd">

Required Attributes

None

Optional Attributes

None

Child Elements

  • <properties> (0 or 1)
  • <classPathEntry> (0..N)
  • <ibatorContext> (1..N)
